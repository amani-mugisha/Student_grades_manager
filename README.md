# Student Grade Manager

## Description
Student Grade Manager is a C++ program that helps manage student grades.  
It stores grades, sorts them, and separates students into pass and fail categories.

##  Features
- Input student grades
- Sort grades in ascending order
- Identify passed and failed students
- Uses C++ STL (vector, algorithm)

## Technologies Used
- C++
- STL (vector, sort)

1. Compile the program:
   ```bash
   g++ main.cpp -o grade_manager
